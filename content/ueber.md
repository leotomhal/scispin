<p class="eyebrow">Über das Projekt</p>
<h1>Beide Extreme<br>sind <span class="turn">Fehler</span>.</h1>

Wissenschaftskommunikation scheitert auf zwei Arten. Sie kann so fachlich und verschachtelt sein, dass niemand sie liest – oder so zugespitzt, dass von der eigentlichen Aussage nichts mehr übrig ist.

Der SciSpin-O-Mat macht beide Richtungen sichtbar. Auf einer siebenstufigen Skala wird dieselbe Meldung von „liest eh keiner" bis „was hat das noch mit der Arbeit zu tun" umgeschrieben. Die Mitte (+1) ist kein Kompromiss, sondern das Ziel: verständlich und korrekt zugleich.

Markierte Stellen zeigen, was sich verändert hat. Gestrichene Passagen – etwa eine weggelassene Einschränkung – werden eigens ausgewiesen, weil gerade das Weglassen oft das Irreführende ist. Jede Stufe bekommt einen kurzen Kommentar, der erklärt, welcher Kommunikationsfehler hier passiert und warum.

Das Werkzeug ist bewusst zugespitzt. Es ersetzt kein Urteil, sondern schärft den Blick dafür, wie Framing eine Aussage trägt oder kippt.
